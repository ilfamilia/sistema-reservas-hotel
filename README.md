# Sistema de Reservas de Hotel

Aplicación de consola desarrollada en Java para gestionar reservaciones de habitaciones de hotel.

## Objetivo
Desarrollar un sistema que permita crear reservas, verificar disponibilidad de habitaciones y validar conflictos entre reservaciones, aplicando principios SOLID.

## Tecnologías
- Java estándar
- Git
- GitHub
